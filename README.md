# CodeAlpha_Credit_Scoring_Mode

## Internship Task - CodeAlpha
This repository contains a machine learning pipeline to predict an individual's creditworthiness based on financial data.

### Approach & Features
* **Algorithm:** Random Forest Classifier.
* **Feature Engineering:** Calculated the Debt-to-Income (DTI) ratio from raw income and debt values.
* **Metrics:** Evaluated the model using Precision, Recall, F1-Score, and ROC-AUC to effectively handle the assessment of financial risk.

### How to Run
`python credit_scoring_model.py`
